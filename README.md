# Fixed APY Staking with Lock Multipliers

## Compile:
```shell
npx hardhat compile
```

## Deploy:
```shell
npx hardhat run scripts/deploy.js
```

## Deploy Factory:
```shell
npx hardhat run scripts/deployFactory.js
```

## Get code coverage:
```shell
npx hardhat coverage
```

## Run tests:
```shell
npx hardhat test
```
